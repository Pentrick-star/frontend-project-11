## Установка и запуск

```bash
npm install
npm run dev
```

## Доступные команды

- `npm run dev` - запуск в режиме разработки
- `npm run build` - сборка для продакшена
- `npm run lint` - проверка кода ESLint
- `npm run sonar` - локальный анализ SonarCloud (требует sonar-scanner)
